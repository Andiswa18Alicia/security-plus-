Match each cryptographic algorithm to its correct category.
Algorithm

Category Definitions
Symmetric Encryption — Same key to encrypt and decrypt
Asymmetric Encryption — Uses a public/private key pair
Hashing — One-way fixed-length output; integrity only
Stream Cipher — Encrypts one bit/byte at a time; deprecated
AES-256 — symmetric block cipher, same key encrypts/decrypts. RSA-2048 — asymmetric, public key encrypts, private key decrypts. SHA-256 — hashing, one-way 256-bit digest, no decryption. RC4 — stream cipher, deprecated (WEP, SSL), broken due to statistical bias.
